Customer Support

MVP User Stories

  1. As a visitor, I want to see the list of flights that are available each day.
  2. As a visitor, I want to create a user account, so that I can make reservations.
  3. As a user, I want to see a profile page that shows how many frequent flyer miles I’ve earned.
  4. As a user, I want to make a reservation.
  5. As a user, I want to see my reservations in my account profile.
  6. As a user, I expect my frequent flyer mileage total to increase every time I reserve a flight.
  7. As a user, I want to search for flights by airport code.
  8. As a user, I want to search for flights by airport code or city name.
  9. As a user, I want to first see my account profile when I login.
  10. As an administrator, I want to view all reservation activity.
  11. As an administrator, I want to see the total amount of revenue to date.
  12. As a user, I want to receive an email confirming my reservation.

Ground Rules (get it?)

  1. The same flights will occur at the same time every single day. A flight is defined as the combination of:
    • A unique flight number
    • Departure airport code
    • Departure time
    • Arrival airport code
    • Arrival time
    • Distance in miles
    • Duration in minutes
  2. Site visitors must create a secure user account in order book a reservation.
  3. A user account should contain at least:
    • First name
    • Last name
    • Email address
    • Secure password
    • Frequent Flyer miles earned
  4. A reservation is defined as the specific combination of: 
    • A user
    • A flight
    • A specific departure date
    • A valid credit card number
  5. Planes fly at about 8 miles per minute, so the arrival time and flight duration could, in theory, be calculated rather than stored in the database.
  6. Every OneWay flight costs the same: $70.
  7. We only accept Visa credit cards. Visa cards are 16-digits numbers that always start with the number 4.
